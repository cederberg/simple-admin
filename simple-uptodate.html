simple-uptodate

Checks if the system requires (security) updates.

Usage:

simple-uptodate [-s] [-m]
Option Description
-s Check only for security updates.
-m E-mail the output to root@localhost instead of the standard output.

Synchronizes the package index with apt-get update and then runs a simulated check for updates and prints the results. Can also filter the updates for security-related issues.

If no updates are available, no output is printed.
