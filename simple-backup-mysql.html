simple-backup-mysql

Performs a MySQL dump of configured databases.

Usage:

simple-backup-mysql [-v]
Option Description
-v Print verbose output.

Performs a MySQL backup with mysqldump of the databases listed in /etc/simple-backup-mysql.conf. The /backup/<hostname>/mysql/ directory is used to store the output MySQL backup SQL files (compressed with lzma).

Backup files are stored for 30 days, except the files from the 1st each month (which are kept indefinitely). The MySQL backup should be run once per day. More frequent runs will overwrite previous backups.

Configuration:

The backup is configured in /etc/simple-backup-mysql.conf with one line per database. Comment or blank lines are ignored. Each line contains the database, user, password and optional command-line switches:

database1 user password
database2 user password --ignore-table=table1
