simple-backup-sync

Performs a remote file sync for configured directories.

Usage:

simple-backup-sync [-v]
Option Description
-v Print verbose output.

Syncronizes one or more directories using rsync. This command will set the appropriate flags to update and remove files on the receiving side, so be careful to sync the correct directories (see rsync).

Configuration:

The syncronization is configured in /etc/simple-backup-sync.conf with one line for each pair of directories. No comment or blank lines are allowed. Each line contains the source dir, destination dir and optional command-line switches to rsync:

/backup/host01 root@host02:/backup/
root@host02:/backup/host02 /backup/ --bwlimit=10
