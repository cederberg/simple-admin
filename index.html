Simple-Admin for Ubuntu

Command-line tools for simplified server administration

Download ZIP Download TGZ View on GitHub

Requirements

  • Ubuntu 10.04 LTS (or later)
  • ...and some command-line tools (auto-installed)

Installation

The install script will install required command-line tools and Perl modules.

  1. Run install script: ./install.sh
  2. Copy & modify config files from etc.

Commands

  • simple-backup-files – Performs a filesystem backup for configured directories
  • simple-backup-mysql – Performs a MySQL dump of configured databases
  • simple-backup-status – Prints info for the backup sync status per machine
  • simple-backup-sync – Performs a remote file sync for configured directories
  • simple-fileinfo – Prints information about a file or command
  • simple-freemem – Cleans filesystem cache (if possible) to recover memory
  • simple-restart – Restarts a group of services in a predefined order
  • simple-status – Checks and prints the machine and process status
  • simple-uptodate – Checks if the system requires (security) updates
  • simple-utf8 – Converts files to UTF-8 (from ISO-8859-1) if needed
  • simple-www-logrotate – Rotates HTTP access log files
  • simple-www-stats – Generates HTTP access log summary statistics
  • simple-zcat – Concatenates and prints compressed files
