Simple-Admin for Ubuntu

Scripts for simplified Ubuntu server administration.

Download ZIP Download TGZ View on GitHub

Requirements

  • Ubuntu 10.04 LTS (or later)
  • ...and some command-line tools (auto-installed)

Installation

The install script will install required command-line tools and Perl modules.

  1. Run install script: ./install.sh
  2. Copy & modify config files from etc.

Commands

  • admin-backup-files – Performs a filesystem backup for configured directories
  • admin-backup-mysql – Performs a MySQL dump of configured databases
  • admin-backup-status – Prints info for the backup sync status per machine
  • admin-backup-sync – Performs a remote file sync for configured directories
  • admin-freemem – Cleans filesystem cache (if possible) to recover memory
  • admin-restart – Restarts a group of services in a predefined order
  • admin-status – Checks and prints the machine and process status
  • admin-uptodate – Checks if the system requires (security) updates
  • admin-utf8 – Converts files to UTF-8 (from ISO-8859-1) if needed
  • admin-www-logrotate – Rotates HTTP access log files
  • admin-www-stats – Generates HTTP access log summary statistics
  • admin-zcat – Concatenates and prints compressed files
