simple-backup-status

Prints info for the backup sync status per machine.

Usage:

simple-backup-status

Prints disk usage and disk size information for the /backup directory. This can be very slow, since calculating disk usage may have to check the size of a very large number of files. An example output:

Backups:
168M    2013-03-19 01:13    /backup/host06
5.4G    2013-03-19 01:13    /backup/host04
5.5G    2013-03-19 01:13    /backup

Filesystem      Size  Used Avail Use% Mounted on
/dev/xvda        24G  7.2G   17G  31% /
