simple-zcat

Concatenates and prints compressed files.

Usage:

simple-zcat <file> ...

Reads <file> and outputs the uncompressed contents to the standard output. If multiple file arguments are present, they are processed in command-line order.

Files with the extensions .bz2, .gz, .lzma or .xz are uncompressed with the corresponding utility. Other files are handled as uncompressed files (with cat). This is similar to zcat (from the zutils package), but supports the .lzma format properly.
