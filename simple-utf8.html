simple-utf8

Converts files to UTF-8 (from ISO-8859-1) if needed.

Usage:

simple-utf8 check <file> ...
simple-utf8 convert <file> ...

The <file> arguments may be either files or directories. All directories will be searched recursively while ignoring any hidden files (.*).

The check command is used to detect text file encoding of text files. The results per file are printed to the standard output.

The convert command is used to convert all text files in the ISO-8859-1 encoding to UTF-8. Information about converted files are printed to the standard output.
