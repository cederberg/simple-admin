simple-backup-search

Searches for matching files in the backup directory.

Usage:

simple-backup-search <pattern>

Searches for all matching file paths in the /backup/<hostname>/files directory. For each file found, the locations to all existing backups are listed.

The search <pattern> matches partial file paths, and may contain wildcards or directory separators. The example below shows the output for simple-backup-search fontconfig:

/etc/java-6-sun/fontconfig.properties: [deleted]
  2010-10-21 00:29      18035  /backup/host06/files/2012-11

/etc/java-7-openjdk/fontconfig.properties:
  2012-10-24 19:00      14538  /backup/host06/files/2013-05
  2012-10-24 19:00      14538  /backup/host06/files/2013-05-22
  2013-04-23 14:03      13959  /backup/host06/files/latest
